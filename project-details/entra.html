Microsoft Entra ID

Microsoft Entra ID is Microsoft's cloud identity and access management service. It helps organizations manage users, groups, applications, authentication, authorization, and secure access to cloud resources.

As a Security Analyst, I use Entra ID to review user access, monitor sign-in activity, support MFA and Conditional Access, and apply least-privilege access principles.

My hands-on experience includes user and group management, role review, sign-in log analysis, access control, MFA enforcement, and identity governance support.

Identity Security

This project demonstrates my practical experience working with Microsoft Entra ID to manage identities, access, and authentication security. The objective was to strengthen identity posture and reduce unauthorized access risk.

I reviewed users, groups, roles, sign-in activity, authentication methods, and access patterns to support secure identity administration.

The project also included applying least-privilege access, reviewing risky sign-ins, supporting MFA adoption, and aligning access controls with cloud security requirements.
